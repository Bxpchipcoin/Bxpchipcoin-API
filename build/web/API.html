Summary

This document provides the complete reference for Bxpchipcoin API.

Bxpchipcoin API has several interfaces to implement them in a custom software:

  • RESTful API that allows:
    • access to the market data: get trades, etc. See Account data RESTful API
    • performing trading operations: get trading balance, get history, etc. See Trading RESTful API
    • managing funds: get balance of the main account, transfer funds between main and trading accounts, create an outgoing transactions, etc. See Payment RESTful API

Trading and payment operations require user’s authentication: each request or message should have a signature.See details in RESTful API authentication and WebSocket API authentication.

Currency symbols

The following currency symbols are traded on Bxpchipcoin exchange.

Symbol Lot size Price step

The actual list of symbols can be obtained by method.

Size representation:

  • Size values in streaming messages are represented in lots.
  • Size values in RESTful market data are represented in money (e.g. in coins or in USD).
  • Size values in RESTful trade are represented in lots (e.g. 1 means 0.01 BTC for BTCUSD)

RESTful API

RESTful API provides the most functional access to Bxpchipcoin facilities. RESTful API allows:

  • access to the market data: trades, etc. See Account data RESTful API
  • performing trading operations: get trading balance,get history, etc. See Trading RESTful API
  • managing funds: get balance of the main account, transfer funds between main and trading accounts, create an outgoing transactions, etc. See Payment RESTful API

Endpoint URL: http://API.bxpchip-coin.org.

Bxpchipcoin provides a demo trading option. You can enable demo mode and acquire demo API keys on the Settings page.
Demo endpoint address: http://API.bxpchip-coin.org

Trading and payment operations require authentication. See also error codes.
